# WebViewPlus

Browser file viewer
(meant to be used with MS WebView2)

Demo: https://mooflu.github.io/WebViewPlus/

## Supported file formats:

### Native via iframe

html, pdf, webp

### Tabular data

csv, xlsx, tsv

### Scalable Vector Graphics (pan & zoom)

svg

### Markdown (with math plugins)

md

### 3D model viewer

gltf, glb, fbx, obj

### Syntax Highlighter

c++, h++, bat, c, cmake,
cpp, cs, css, go, h, hpp,
java, js, json, jsx, lua,
perl, pl, ps1, psm1, py,
rb, sass, scss, sh, sql,
tex, ts, tsx, txt, yaml, yml

## Develop

npx vite

npx vite build

To test release build with firefox set:
privacy.file_unique_origin false
